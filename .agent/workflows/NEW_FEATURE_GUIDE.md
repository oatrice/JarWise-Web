# ✨ New Feature Guide

ขั้นตอนการเพิ่มฟีเจอร์ใหม่ในระบบ JarWise

## 1. Define Requirements (Shared Logic)
- กำหนด Business Logic
- วางแผน Data Model

## 2. Implement Web (Prototype)
- เริ่มที่ **Web** (`/Web`) ก่อนเสมอ
- สร้าง Mock Data ใน `src/utils/mockData.ts`
- สร้าง UI Component ใหม่

## 3. Implement Mobile/Native
- นำ Design จาก Web ไปทำต่อใน **Mobile** หรือ **Android**

## 4. AI & Automation (Antigravity)
- ใช้ **Antigravity Skills** ช่วย Gen Code/Docs
